# What's Markdown?

- A simple, human-readable markup language for creating formatted text
- Really family of languages
	- [CommonMark](https://spec.commonmark.org/)
	- [GitHub Flavored Markdown](https://github.github.com/gfm/)
	- Pandoc
	- ...
- Common(Mark) features:
	- Headings
	- List (numbered/unnumbered) and quote blocks
	- Links
	- Images
	- Text formatting, such as *emphasis*
	- Raw HTML
