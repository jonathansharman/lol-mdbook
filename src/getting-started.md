# Getting Started

1. Install `mdBook` by either...
	- Downloading a binary from the [releases page](https://github.com/rust-lang/mdBook/releases)
	- [Installing Rust](https://www.rust-lang.org/tools/install) and then
	  running `cargo install mdbook`
2. Create a new book with `mdbook init <title>`
3. Run `mdbook serve` to serve the book, detect changes, and rebuild
4. Write the book!
